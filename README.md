# Installation
1. Add the main application

    ```bash
    git clone https://github.com/YeeJiaWei/manzana.git
    ```
2. Add the device key

    [here](https://github.com/FreeL3CDM/FreeL3CDM)

3. Build the container

    ```bash
    docker compose up
    ```

# Usage
```bash
python3 manzana.py
```
